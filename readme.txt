
Proyecto Final Desarrollo WEB

El proyecto consta de una pagina de inicio (index.html), tres paginas (quiensoy.html, proyecto.html y contacto.html).
Estas cuatro paginas se vinculan entre si mediante  una barra de navegacion hecha con Bootstrap.
A su vez, se accede a una quinta pagina (prensa.html) desde la pagina proyecto.html mediante botones.

Este proyecto sufrio modificaciones conforme iban pasando las clases. Una vez terminadas, me intereso mucho el uso de CSS grid
,asi que decidi rehacer todo el proyecto utilizando este sistema, pero, manteniendo la estetica vista con el tutor.
Use, segun el caso, GRID AREAS o FILAS Y COLUMNAS EXPLICITAS.
Esta ultima me resulto muy comoda para trabajar los tamaños mas pequeños.
En cuantos a los puntos de interrupcion, he utilizado los de bootstrap (<576px, 768 px, 992px y mas de 1200px).

He intentado incorporar al proyecto la mayor parte de los conceptos aprendidos 
Flex-box para manejar los elementos en el interior de la grid, mixin, import, extend, maps, etc.

Tambien incorpore sass / scss a todo el proyecto y aplique SEO a todas las paginas del proyecto.

Por ultimo, para el carrousel del index, utilice bootstrap.
